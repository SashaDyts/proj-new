HTML-фрагменты

Плагин posthtml-include позволяет добавлять в HTML-файлы страниц код из других HTML-файлов (фрагментов). Фрагменты хранятся в папке src/partials. Добавление происходит при компиляции файлов страниц, например index.html.

Этот файл example.html — пример такого фрагмента, с разметкой определённой части главной страницы. Lorem ipsum dolor, sit amet consectetur adipisicing elit. Quis cupiditate odit numquam maiores sed, molestiae nam est possimus perferendis dolore quasi expedita porro deserunt laudantium mollitia, aspernatur saepe, ab consequuntur.

Синтаксис добавление фрагмента в главный файл: <include src="./partials/имя-файла.html"></include>
